### Module Mapping Table
| Module/Project Name | Directory Path | Key Responsibility | Entry Point |
| :--- | :--- | :--- | :--- |
| **Main UI (EXE)** | `/src/[AppName]` | MFC Application Entry & UI Controllers | `[App]App.cpp`, `[App]Dlg.cpp` |
| **Vision (DLL)** | `/src/Vision` | Vision Processing & Inspection Algorithms | `Vision.cpp`, `VisionProcessor.cpp` |
| **Hardware (DLL)** | `/src/Hardware` | Camera, PLC, Motion Control Interfaces | `HardwareManager.cpp` |
| **Common (DLL)** | `/src/Common` | Shared Utilities & Data Structures | `CommonUtil.cpp`, `Types.h` |

## 3. KEY FILE PATTERNS

### File Naming Conventions
| Pattern | Purpose | Location |
| :--- | :--- | :--- |
| `*App.cpp` | MFC Application class | `/src/[MainApp]/` |
| `*Dlg.cpp` | Dialog classes | `/src/[MainApp]/` |
| `MainFrm.cpp` | Main frame window | `/src/[MainApp]/` |
| `*View.cpp` | MFC View classes | `/src/[MainApp]/` |
| `*Doc.cpp` | MFC Document classes | `/src/[MainApp]/` |
| `*Processor.cpp` | Core processing logic | `/src/Vision/` or `/src/[Module]/` |
| `*Service.cpp` | Service layer classes | `/src/[Module]/` |
| `*Manager.cpp` | Resource managers | `/src/[Module]/` |
| `*Data.h` | Data structures | `/include/Common/` |
| `*Def.h` | Definitions & constants | `/include/[Module]/` |
| `*Types.h` | Type definitions | `/include/Common/` |

### Resource Files
| Pattern | Purpose | Location |
| :--- | :--- | :--- |
| `*.rc` | Resource scripts | `/res/` or `/src/[MainApp]/` |
| `resource.h` | Resource ID definitions | `/src/[MainApp]/` |
| `*.ico` | Icons | `/res/Icons/` |
| `*.bmp` | Bitmaps | `/res/Bitmaps/` |

## 4. SEARCH & NAVIGATION RULES

### Search Priority Workflow
1. **Check this file**: Identify target module from table above
2. **Navigate to target directory**: Go directly to specific folder
3. **Targeted search**: Search within that directory only
4. **Report location**: Confirm file path before proceeding

### Search Examples
| User Request | AI Action |
| :--- | :--- |
| "Find the main dialog class" | Search `/src/[MainApp]/` for `*Dlg.cpp` |
| "Locate vision processing code" | Search `/src/Vision/` for `*Processor.cpp` |
| "Where is the camera interface?" | Search `/src/Hardware/` for `Camera*.cpp` |
| "Find common type definitions" | Search `/include/Common/` for `Types.h` or `*Def.h` |

### Prohibited Actions
- ❌ **No Global Search**: Do not use `grep -r` or `find / -name` from root
- ❌ **No Recursive Scans**: Do not search all subdirectories unless explicitly requested
- ❌ **No Assumptions**: Do not guess file locations without consulting this mapping

### Dependency Check Protocol
Before modifying any header file:
1. Identify all `.vcxproj` files that include it
2. Identify all `.cpp` files that `#include` it
3. Report potential impact to user
4. Get approval before modifying
