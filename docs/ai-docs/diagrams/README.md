# Architecture Diagrams

AI-generated mermaid diagrams. Created during Project Discovery, updated when
architecture changes.

All diagrams use [Mermaid](https://mermaid.js.org/) syntax — text-based,
git-friendly, renderable in GitHub/VS Code.

## Files

| Diagram | Purpose | When to update |
|:---|:---|:---|
| `dependency-graph.md` | Module/package dependency relationships | New module added or dependency changed |
| `architecture-layers.md` | System layer overview | Layer structure changed |
| `data-flow.md` | Data flow through the system | Processing pipeline changed |

## Viewing

- **GitHub**: Renders mermaid blocks natively in `.md` files.
- **VS Code**: Install "Markdown Preview Mermaid Support" extension.
- **CLI**: Use `mmdc` (mermaid-cli) to export to PNG/SVG.
